# 贷款利息计算器

## 三种计算场景

### 场景一：等额本息分期贷款利息

公式：

```
总利息 = (还款期数 + 1) × 贷款总额 × 期利率 ÷ 2
总还款额 = 贷款总额 + 总利息
每期还款额 = 总还款额 ÷ 还款期数
```

利率档次：
- ≤1 年：4.35%
- 1–5 年：4.75%
- >5 年：4.90%

期利率 = 年利率 ÷ 每年还款次数

### 场景二：短期资金占用利息（首付/延期支付）

适用于提车后 N 个月才支付的款项，按期限匹配利率：

```
利息 = 金额 × 年利率 × (延迟月数 / 12)
```

利率按延迟月数所在档位确定。

### 场景三：承兑汇票贴现利息

适用于以银行承兑汇票支付的款项：

```
利息 = 金额 × 贴现年利率 × (票据天数 / 360)
```

贴现年利率：银行承兑汇票统一按 3.5%；票据天数默认 90 天（3 个月承兑）或 180 天（6 个月承兑，少见）。

### 场景四：车款综合计算（首付 + 分期 + 承兑）

金龙客车常见付款模式：首付（延期支付）+ 剩余车款分期 + 全部款项以承兑汇票支付。

计算步骤：
1. 贷款总额 = 总车款 − 首付款
2. 场景二：首付资金占用利息（按首付延迟月数）
3. 场景一：分期贷款利息（按贷款总额、年限、半年供）
4. 场景三：全部 N 笔（首付 + 各期）承兑贴现利息

汇总 = 首付占用利息 + 分期利息 + 承兑贴现利息

> 首付占用利息与分期利息为内部资金成本核算，客户不承担。承兑贴现为实际成本，根据合同约定确定承担方。

## 使用方式

### 在对话中直接计算

1. 识别用户描述中的计算场景
2. 提取金额、期限、还款频率
3. 匹配对应利率档次
4. 代入公式分步计算，展示过程
5. 汇总输出

优先级：用户明确说"按等额本息""按skill公式"时，使用场景一公式；说"计算承兑"时，使用场景三。完整车款描述时，执行综合计算。

## 注意事项

- 等额本息公式为利息近似计算，非精确 IRR
- 利率以中国人民银行贷款基准利率为准
- 半年供：每年还款 2 次，期利率 = 年利率 ÷ 2
- 承兑贴现天数按 360 天/年基准计算
- 金额保留两位小数，单位统一为人民币（元）
